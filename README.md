# 📡 Networking Lab Sheets

A collection of hands-on networking lab configurations and simulation files completed as part of the **BICT (Hons) — Computer Networking Technology** pathway at the **University of Kelaniya, Sri Lanka**.

All Packet Tracer (`.pkt`) files are built and tested using **Cisco Packet Tracer**, covering foundational to intermediate networking concepts.

## 🧪 Labs Covered

| Lab | Topic | Tools Used |
|-----|-------|------------|
| Ethernet Switching | MAC address tables, switch forwarding | Cisco Packet Tracer |
| Static Routing | Manual route configuration between routers | Cisco Packet Tracer |
| VLANs & Inter-VLAN Routing | VLAN segmentation, trunk links | Cisco Packet Tracer |
| Inter-VLAN via SVI | Layer 3 switching using SVIs | Cisco Packet Tracer |
| VLANs, VTP & DTP | VLAN propagation, trunk negotiation | Cisco Packet Tracer |
| VirtualBox Introduction | VM setup for network simulation environments | VirtualBox |

---

## 🛠️ Tools & Technologies

- **Cisco Packet Tracer** — Network simulation
- **Oracle VirtualBox** — Virtualization environment
- **Cisco IOS CLI** — Switch & router configuration

---

## 📚 Module Context

These labs are part of the networking modules in the **Faculty of Computing and Technology**, University of Kelaniya, aligned with **Cisco Networking Academy** curriculum standards.
